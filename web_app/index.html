Here2Serve

This is the individual project for the Internet of Things course of the a.y. 2021-2022 at "Sapienza" university. The system is created to serve the user some drinks whenever requested. Here is shown a dashboard that contains informations regarding the use of the system. More specifically, we can see the time where the user has used our system and the over all usage of it. On the right we can see some statistics from the overall usage of the system. For more information regarding to the system, here is the link for the github_repository

The chart shows the hour and minute as well as the overall time expressed in seconds the system is used
